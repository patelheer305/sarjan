How to Use This App

  1. Select your preferred measurement layout from the Unit System menu and enter the Saw Kerf (cutting blade width).
  2. Input the available inventory dimensions and counts under the Stock Materials panel.
  3. Enter your required specific target cutting sizes and amounts into the Required Cuts section.
  4. Click Run Optimization to instantly calculate the absolute most efficient layout blueprint to heavily minimize raw scrap waste!

Privacy Policy

Last updated: June 2026

Welcome to Sarjan. We highly value your trust and are fully committed to protecting your personal information. This policy explicitly documents our privacy operations across deployments, including the Google Play Store.

1. Information Collection and Use

Sarjan is engineered to run zero external pipeline processes:

  • No Personal Data: The application does not collect, track, query, or store identities, account logs, or contact records.
  • Local Computations: Your inventory matrices, measurements, cutting details, and configuration figures are completely calculated inside your machine sandbox.
  • Device Cache Memory: Configuration variables cache straight to local context hardware storage (localStorage) strictly to prevent input data dropouts upon system context updates or app resets.

2. PDF Generator Assets

Exported data sheets assemble directly into document memory layers client-side. No calculation matrices or project values migrate onto outside host elements to finalize structural documents.

3. Play Store Service Modules

Core binaries function inside Google system containers. System analytics run securely through target infrastructure packages for basic app crash logging and performance checking. Review comprehensive structures at the Google Privacy Policy tracking space.
